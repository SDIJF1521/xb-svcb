## v0.0.8 · 合唱混声 + 可视化时间轴基础版 + 后台传输

### 新增
- 🎤 **混合翻唱支持「合唱」**：一句歌词可同时指派多个模型，多路人声去相关叠加（恒功率声像铺开 + 微失谐 detune + Haas 微延迟 + 软限幅防破音），听感为真正的多人合唱而非「一个更响的声音」。
- 🧬 **基础版可视化时间轴**：多模型混合页新增时间轴总览，按歌词时间把每句渲染为横向色块；色块宽度对应句子时长，单模型显示模型主题色，合唱句显示多模型渐变与人数角标，未指派片段显示为间奏。
- 🖱️ **时间轴点击指派**：点击时间轴色块可直接弹出模型选择面板，为该句增减演唱模型；时间轴与逐句列表共享同一份指派状态，修改后实时同步。
- 🌐 **模型站上传 / 下载挂后台**：传输不再阻塞前端，可继续浏览操作；顶栏新增「传输」面板，统一查看上传 / 下载实时进度，支持清除已完成。
- 📑 **分页「加载更多」**：模型站搜索与在线资源获取（网易云 / QQ音乐）均改为分页加载，减少单次查询等待。

### 优化
- 🎨 **逐句指派 UI 重做**：被指派模型以彩色胶囊呈现（主题色圆点 + 同色描边），合唱句带「🎤 合唱 ×N」徽标，圆形「+」弹出模型选择面板，多选即合唱，直观清晰。
- 🧭 **导航栏简化**：选中项采用主色文字 + 渐变下划线，去除冗余装饰，更克制现代。
- 🎚️ **合唱听感增强**：相比单纯叠加，多模型同唱时会自动做声像展开、微失谐与微延迟去相关，减少「只听到一个声音」的问题。

### 说明
- 当前时间轴为**基础版总览**：支持按歌词时间展示片段、点击片段指派模型、合唱可视化；暂不包含拖拽边界、缩放编辑、波形显示与自动切句，这些将作为后续可编辑时间轴能力继续迭代。

### 安装
- 下载并运行 `XB-SVCB-Setup.exe` 一键安装（约 2 GB，含分离 / so-vits-svc / RVC / 模型站组件）。
- GPU 加速需 NVIDIA 显卡 + CUDA 12.1 运行库：https://developer.nvidia.com/cuda-12-1-0-download-archive
