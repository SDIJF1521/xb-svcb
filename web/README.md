# XB-SVCB 前端

版本：`0.0.22`

这里是 XB-SVCB 桌面应用的 Vue 3 + Vite 前端。生产构建产物会由 `installer/xb-svcb-app.spec` 打进 `XB-SVCB.exe`。

## 开发

```sh
npm install
npm run dev
```

## 构建

```sh
npm run build
```

构建输出目录是 `web/dist`，随后由桌面本体打包流程内置进应用。

## v0.0.22 重点

- 前端版本同步到 `0.0.22`，与应用本体、Python 锁文件和安装器版本一致。
- 创建翻唱与编辑器设备选项由后端环境能力动态生成，按实际环境显示 NVIDIA CUDA、AMD ROCm、AMD DirectML 或 CPU。
- 创建翻唱会取 UVR 与当前歌声模型框架共同支持的设备，避免选择只在其中一个隔离环境可用的加速器。
- 环境或模型框架变化后会自动修正失效的旧设备偏好，并显示当前后端与显卡名称。

## v0.0.21 重点

- 前端版本同步到 `0.0.21`，与应用本体、Python 锁文件和安装器版本一致。
- 插件选择器当前面向任意目录中的 64 位 Windows `.vst3` 音频效果器；不支持 VST2 `.dll`、32 位插件、CLAP、AAX、AU 或需要 MIDI 音符的 VST3i 乐器。
- 插件控制弹窗改为非模态、可拖动且允许点击底层编辑器；原生 VST3 GUI 打开后可继续操作播放与时间轴。
- 插件播放位置以 `120ms` 周期同步给 JUCE Host；原生会话回传实际采样位置、结束状态、声卡缓冲和延迟，时间轴跟随真实输出位置。
- 混音播放使用全局播放位置驱动插件监听；单片段播放只在当前片段属于已打开插件时送入监听，避免错误素材驱动插件仪表。
- VST3 GUI 会话握手成功时切换到 JUCE 块级可听输出，插件旋钮在下一音频块生效；无原生设备时继续使用双 HTML Audio 约 `90ms` 交叉淡化回退。
- 乐器/MIDI 或无音频总线插件会显示兼容提示并保留原播放器；运行中触发静音/非法采样保护时，面板与全局消息会提示已自动旁通为干声。
- 切换原生实时会话时会无条件停止两个 HTML Audio 输出；原生 Host 回传播放头就绪状态并向依赖时间上下文的插件提供曲线/节拍同步数据。
- 播放期间只同步轻量宿主状态，暂停或关闭时再保存参数/state，避免大型插件频谱在调参或工程同步时停止刷新。
- 片段音量、淡入淡出、音量包络和内置效果参数在拖动时以 `120ms` 防抖保存并触发播放热更新。
- 编辑器新增“合并下一段”，把所选片段与同轨后一片段渲染为一个新音频并保持可撤销。

## v0.0.20 重点

- 前端版本同步到 `0.0.20`，与应用本体、Python 锁文件和安装器版本一致。
- 模型导入、模型管理、创建任务和编辑器局部重推理支持 DDSP-SVC 6.3。
- DDSP-SVC 参数面板提供 Rectified Flow 采样步数、F0、设备和 `-2.00` 至 `+2.00` 半音共振峰偏移。
- 消息状态集中到全局 Pinia store，作品和系统状态更新可跨页面同步，已读状态不再由单个顶栏组件独占。
- 编辑器歌词入口支持 TXT / LRC；无时间戳 TXT 显示“自动切句”，LRC 保留工程时间和片段时间选择。
- 多角色管理支持角色增删、模型关联、变调、颜色、备注和片段分配；时间轴模板提供独唱、对唱、主唱和声与三角色剧情结构。
- UVR 设备选择与推理参数保持一致，任务日志可显示实际分离与去混响设备。
- 暗色与亮色切换优先使用 WebView2 原生页面快照，圆形过渡采用平滑减速，并在动画结束后同步原生标题栏；旧内核仍使用带柔化边缘的兼容动画。

## v0.0.19 重点

- 前端版本同步到 `0.0.19`，与应用本体和安装器版本一致。
- 音频编辑器播放中修改效果器、参数、音量包络或其他声音内容时，会在保存后防抖请求新的混音预览。
- 旧预览在后台渲染期间继续播放；新预览就绪后按当前时间点热切换，并继续保持播放状态。
- 混音内容签名会过滤工程标题、锁定状态等非声音修改，避免无意义的预览重载；过期请求不会覆盖较新的效果状态。
- 后台更新期间播放按钮仍可立即暂停；渲染失败时保留当前预览并显示错误提示。
- 时间轴播放头使用标尺真实左边界作为零点，修复原先轨道标题宽度与硬编码偏移不一致造成的 `32px` 固定误差，并兼容横向滚动。
- 在线歌词 UI 继续使用统一的歌词结果类型，网易官方嵌套歌词地址与 QQ 聚合歌词回退由桌面音乐服务完成。

## v0.0.18 重点

- 前端版本同步到 `0.0.18`。
- 模型导入、模型管理、创建任务和模型站 UI 支持 SeedVC，并提供目标音色参考音频选择。
- 后台模型下载完成后立即刷新可选模型；下载失败会显示原因和“继续下载”，完成后可直接跳转查看本地模型。
- 自定义主题背景支持图片和 MP4 动态壁纸，桌面端持久化媒体，浏览器开发模式可直接预览。
- 在线曲库 UI 移除上游不再支持的每页数量与加载更多，展示曲库名称和实际返回数量。
- QQ音乐试听优先尝试会员音频地址，并保留普通地址回退。

## v0.0.17 重点

- 前端版本同步到 `0.0.17`。
- 音频编辑器新增效果器面板能力：音量包络、内置效果器参数、插件路径选择和 JUCE VST3 插件窗口入口。
- 新增片段/音轨音频复制入口、工具栏/轨道级粘贴入口，以及 `Ctrl/Cmd + V` 从系统剪贴板粘贴音频到播放头的快捷操作。

- 前端 API 新增 JUCE Host 状态检查、插件检查、插件 GUI 打开/关闭、插件 state 同步、音频复制和剪贴板粘贴结果类型。
- 局部重推理完成后会提示被移除的插件类效果，避免用户误以为 VST3 处理已经写入新的 AI 干声。

## v0.0.16 重点

- 主题切换改为从顶栏主题按钮扩散的圆形过渡，切换暗色、亮色和自定义主题时保留旧主题作为圈外画面。

## v0.0.15 重点
