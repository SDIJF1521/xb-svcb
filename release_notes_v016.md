## v0.0.16 · 主题系统升级 + 多角色时间轴 + GPU 环境保护

### 新增

- 🎨 **自定义主题编辑器**：用户可以编辑主题色、文字色、边框、阴影和页面背景，支持添加背景图片并开启动态粒子。
- 🌤️ **亮色默认示例**：自定义主题默认提供「晴空花园」配色，作为更容易继续调整的亮色模板。
- 🎭 **多角色管理**：音频编辑器新增角色列表，可配置角色名称、颜色、模型、变调和备注，并把选中的片段分配给指定角色。
- 🧩 **时间轴模板**：新增独唱、对唱、主唱 + 和声、三角色剧情等模板，快速生成角色、音轨和示例片段结构。

### 优化

- 🌓 **主题切换动画**：主题切换改为从顶栏主题按钮触发的圆形扩散/收回效果，切换时圈外保留旧主题画面，体验更接近 Element Plus 官网。
- 🧱 **前端组件化**：主题相关逻辑拆分到 `ThemeSwitcher`、`ThemePresetList`、`CustomThemeEditor`、`ThemeBackground` 等组件；编辑器角色和模板也拆成独立组件，便于继续扩展。
- 🎛️ **编辑器可读性**：角色颜色会同步到片段和音轨标识上，复杂多人段落更容易定位和调整。
- 📦 **版本统一**：前端包、应用本体、安装器脚本和锁文件版本同步更新为 **v0.0.16**。

### 修复

- 🛠️ **修复 GPU 用户部署 UVR 环境时被降级到 CPU torch 的问题**：安装脚本会在 UVR/RVC/SVC 环境搭建时保护已选择的 GPU PyTorch 栈，避免兼容 NVIDIA 显卡被错误替换为 CPU 版 PyTorch。
- ✨ **修复主题切换闪烁和圈外黑屏感**：暗色主题与自定义主题切换时，扩散圈外不再显示全黑底，页面组件在过渡期间保持可见。

### 安装

- 下载并运行 `XB-SVCB-Setup.exe` 一键安装。
- 前端、应用本体与安装器版本已同步更新为 **v0.0.16**。
- 40 系及以下兼容 NVIDIA GPU 默认使用 cu121 栈；50 系（Blackwell）使用 cu128 栈；CPU 或不兼容显卡才会使用 CPU torch。
