## v0.0.20 · DDSP-SVC 6.3 + 全局消息同步 + 编辑器工作流

> 本次更新完成 DDSP-SVC 6.3 从模型导入到成品混音的完整链路，统一消息状态与设备选择，补齐 TXT 歌词、多角色和时间轴模板工作流，并优化暗色主题切换体验。

> [!IMPORTANT]
> v0.0.20 继续使用分卷安装包。请同时下载 `XB-SVCB-Setup.exe` 和全部 `XB-SVCB-Setup-*.bin`，并放在同一目录后再运行安装程序。

### 🎙️ DDSP-SVC 6.3

- 📥 **模型导入与独立管理**：支持导入 Rectified Flow 的 `model.pt` 与 `config.yaml`，并在模型库中按 `ddsp-svc` 框架管理。
- 🔀 **完整推理路由**：新增 DDSP-SVC Engine 与隔离 worker；创建页、多模型混唱和编辑器局部重推理均按模型框架自动路由。
- 🎚️ **专属推理参数**：支持变调、RMVPE / CREPE / Harvest / Parselmouth、说话人 ID，以及 `10` 至 `50` 步 Rectified Flow 采样。
- 🗣️ **共振峰偏移微调**：提供 `-2.00` 至 `+2.00` 半音、步进 `0.05` 的自定义参数，映射上游 `formant_shift_key`；仅 pitch augmentation 模型有效。
- 🎼 **高音域模型兼容**：worker 按模型 YAML 读取 `f0_min` / `f0_max`，不再强制使用上游默认音域。
- 🧠 **完整底模链路**：使用 ContentVec、RMVPE 与 PC-NSF-HiFiGAN 2025.02；PC-NSF-HiFiGAN 随安装器离线提供，缺失时才联网获取。
- 🧰 **安装步骤修复**：补齐 DDSP 底模检查所需的 `json` 模块导入，修复依赖安装完成后出现 `name 'json' is not defined` 的问题。

### ⚙️ UVR 与设备

- 🎮 **严格遵循设备选择**：UVR 分离与去混响继承任务选择的 `cuda` / `cpu`，不再无提示切换设备。
- 🚫 **禁止 GPU 静默回退**：选择 CUDA 但当前环境不可用时给出明确错误，不再悄悄使用 CPU 完成任务。
- 🔍 **记录实际运行设备**：任务日志显示实际分离设备和去混响设备，便于确认 GPU 是否真正生效。
- 🟢 **匹配 NVIDIA Torch 栈**：40 系及以下使用 `torch 2.5.1+cu121`，RTX 50 系 Blackwell 使用 cu128 栈。
- ✅ **安装后 CUDA 复核**：UVR 与 DDSP-SVC 环境均验证 `torch.cuda.is_available()`；校正或驱动异常时明确失败，不再误报 GPU 环境就绪。

### 🔔 消息中心

- 🌐 **全局状态同步**：消息状态迁移到全局 Pinia store，作品进度、任务完成/失败和系统状态可跨页面持续同步。
- 🪟 **多窗口已读同步**：通过 BroadcastChannel / localStorage 同步已读状态，不再因顶栏重建而重复或丢失通知。
- 🔄 **轮询请求合并**：作品 store 合并重复状态请求并维护任务终态，减少页面切换造成的状态断层。

### 🎼 音频编辑器

- 📝 **TXT / LRC 歌词导入**：支持 UTF-8 BOM、UTF-8 和 GB18030；LRC 继续按工程时间或片段时间中的时间戳切分。
- ✂️ **TXT 自动切句**：按每行或中文 / 英文句末标点识别歌词句子，并自动生成时间轴切点。
- 🤫 **静音辅助对齐**：自动切句优先吸附片段中的静音中心；没有合适静音时，按每句歌词长度比例分配时间。
- 👥 **多角色管理**：支持角色增删、颜色、关联模型、变调、备注和选中片段分配，角色数据随编辑工程持久化。
- 🧩 **时间轴模板**：内置独唱、双人对唱、主唱和声与三角色剧情模板，模板轨道和角色映射随工程保存。

### 🌗 主题体验

- 🌑 **暗色切换更自然**：WebView2 优先使用原生页面快照，圆形扩散采用平滑减速，不再使用容易错位的实时 DOM 克隆作为首选路径。
- 🪟 **原生窗口延后同步**：标题栏和窗口边框在网页动画完成后再切换颜色，避免页面尚未完成过渡时窗口外观提前跳变。
- 🧭 **兼容路径修复**：旧内核的备用动画保留当前滚动位置，并使用柔化边缘降低暗色与亮色之间的突兀感。

### 🛠️ 修复

- 修复 DDSP 安装脚本缺少 `json` 导入，导致底模配置检查必然失败的问题。
- 修复项目内已有 PC-NSF-HiFiGAN、安装器却仍尝试从 GitHub Release 下载的问题。
- 修复 UVR 任务选择 GPU、实际环境却是 CPU Torch 时继续静默运行的问题。
- 修复任务状态依赖顶栏组件生命周期，切换页面后通知可能中断或重复的问题。
- 修复暗色主题过渡在滚动页面和复杂控件下出现画面错位、动画后段突然加速的问题。

### ⚠️ 兼容与性能

- DDSP-SVC 当前实测可用显存下限约为 **4.5GB**；建议至少 **6GB**，推荐 **8GB**。4GB 显卡只建议尝试短音频或改用 CPU。
- DDSP-SVC 会按静音切片逐段推理，峰值显存取决于最长连续非静音片段；Rectified Flow 采样步数主要影响耗时。
- 本版本不导入或解析 SVCFusion `.sf_pkg` 工程包；DDSP 模型继续使用 checkpoint 与 YAML 原始文件导入。
- 共振峰偏移仅对使用 pitch augmentation 训练的模型有效；普通模型即使设置该参数也不会得到预期效果。
- 已安装旧版可覆盖升级；模型、作品、下载素材、编辑工程、设置、消息已读状态和主题媒体继续使用原数据目录。
