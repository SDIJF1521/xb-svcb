## v0.0.23 · FastAPI 外部接入

> 本次更新为 XB-SVCB 增加可由外部程序调用的 FastAPI 服务。服务默认关闭，只能由用户在软件内手动启动；关闭软件后不会继续监听端口。v0.0.22 的 NVIDIA CUDA、AMD DirectML 与 CPU 推理环境保持原有行为。

> v0.0.23 继续使用分卷安装包。请同时下载 `XB-SVCB-Setup.exe` 和全部 `XB-SVCB-Setup-*.bin`，放在同一目录后再运行安装程序。

### FastAPI 服务

- 新增“资料库 -> API 接入”页面，可选择仅本机 `127.0.0.1` 或局域网 `0.0.0.0` 监听，并配置 `1024–65535` 端口。
- 服务不会随软件启动而自动运行；每次打开软件后需要用户手动点击“启动服务”。退出软件会停止 Uvicorn 并释放端口。
- FastAPI 在 `XB-SVCB.exe` 当前 GUI 进程的后台线程运行，不启动 `cmd.exe`、PowerShell 或其他可见控制台窗口。
- API 与桌面界面复用同一套模型服务、作品仓储和串行推理队列。外部创建的任务可在“我的作品”查看，界面任务也可通过 API 查询。
- 支持跨域请求，便于局域网内的浏览器工具或其他桌面程序接入。

### 接口与鉴权

- `GET /health` 提供无需密钥的健康检查；其余 `/api/v1` 接口统一使用 `X-API-Key`。
- API 接入页可查看、复制和重新生成密钥。服务运行期间禁止修改端口、监听范围或密钥，避免调用配置与实际服务不一致。
- 提供系统环境、模型列表、单个模型、音频上传、上传清理、任务创建、任务列表、队列、任务进度、重试与生成音频下载接口。
- 模型和任务响应使用单独的 HTTP DTO，不返回本机模型权重、配置、源音频、日志等内部绝对路径。
- SeedVC 外部任务支持单独上传目标音色参考音频，通过 `reference_upload_id` 传入。

### 大文件上传

- 上传接口不设置文件大小上限，使用固定小块流式读取并写入当前数据目录，不会把完整音频一次载入内存。
- 实际可上传容量只受数据目录所在磁盘剩余空间及调用方网络超时设置限制。
- 支持 WAV、FLAC、MP3、M4A、AAC、OGG、Opus、WMA 及常见视频容器；上传文件保存在 `api/uploads` 并随数据目录迁移。
- 数据目录迁移会在 API 服务或推理队列运行时阻止执行，避免上传、任务和配置在迁移过程中指向不同目录。

### 文档与测试

- 软件内置 API Key 鉴权连通性测试，会实际调用模型接口并显示延迟和模型数量。
- 服务启动后可直接打开 Swagger UI `/docs`、ReDoc `/redoc` 与 OpenAPI JSON `/openapi.json`。
- Swagger 中每个接口按 go-cqhttp 风格展示用途、method、path、鉴权、参数表、请求示例、返回示例和响应字段表；创建任务页逐项说明四种模型的全部推理参数。
- Swagger 页面顶部提供可直接运行的 Python 完整流程，覆盖健康检查、上传歌曲、读取模型、SeedVC 参考音频、提交任务、轮询进度、流式下载成品和清理上传文件。
- API 接入页提供可复制的 Python `requests` 与 PowerShell 7 完整示例，覆盖上传、选模型、提交、轮询和下载。
- 安装目录新增 `docs/api.md`，可离线查看安全配置、标准流程、SeedVC 用法、接口清单与常见状态码。

### 安装与发布

- 应用本体新增 `fastapi`、`uvicorn` 与 `python-multipart` 依赖；PyInstaller 显式收集 Uvicorn 动态 HTTP 协议、事件循环和 lifespan 模块。
- 应用、Python 项目、前端包、两份锁文件、Windows EXE 版本资源和 Inno Setup 均同步为 `0.0.23`。
- 安装器默认不创建 API 自动启动项、不开放固定防火墙规则，也不改变现有 CUDA、DirectML 或 CPU 环境搭建逻辑。
- 发布构建会校验 `release_notes_v023.md` 与 `docs/api.md`，安装时将两份文档一并释放到安装目录。

- 服务生命周期测试覆盖默认关闭、真实端口启动、鉴权连通性测试、停止服务和端口释放。
- 前端 TypeScript 检查与生产构建覆盖 API 管理页、桌面桥方法、浏览器 mock 和响应式布局。
