## v0.0.9 · 可编辑可视化时间轴 + 下载可播放校验 + LRC 导入

### 新增
- 🎚️ **可编辑可视化时间轴**：混合翻唱时间轴升级为可编辑——
  - **拖动边界**：拖拽色块左右边缘调整片段起止，自动**吸附**到歌词时间 / 相邻片段边界 / 首尾。
  - **缩放**：横向放大时间轴局部精修，刻度随缩放自适应密度。
  - **拆分 / 合并 / 删除**：从中点拆分片段、与后段合并（并集模型 + 文本拼接）、一键设为间奏或删除。
- 🧩 **片段模型独立化**：片段与歌词**解耦**，每段拥有独立的起止时间与模型指派；歌词仅作为初始切分与吸附锚点，编辑后不再受歌词行约束。
- 📥 **导入本地 `.lrc` 歌词**：在线获取之外新增「导入歌词文件」，支持带时间轴的 LRC（兼容一行多时间标签、忽略元数据），导入即生成可编辑片段；原有在线获取方式保持不变。
- 🔊 **下载前可播放校验**：在线资源下载会校验是否为可播放音频——按「文件头魔数 > Content-Type > ffprobe 实测解码」判定，VIP / 无版权 / 失效链接（返回错误页或无法解码）**不可下载**，并在列表标记「不可播放」、禁用其下载按钮，避免坑到后续推理。

### 优化
- 🎨 **合唱多模型 UI 重构（不再撑破布局）**：逐段模型胶囊超过 3 个折叠为「+N / 收起」，时间轴色块仅显示合唱人数角标，指派弹窗与模型列表可滚动，多模型合唱场景下界面不再横向溢出。
- ⏱️ **整体偏移按增量平移**：调整整体偏移时按增量平移所有片段，保留已做的拆分 / 合并 / 边界编辑，不再清空。
- 🪟 **放大编辑改为弹窗**：时间轴精修在独立弹窗中进行，内联仅保留固定高度缩略预览，编辑不再撑破主界面布局。

### 修复
- 🩹 **拖动拉伸「越拖越长 / 突然放大撑破 UI」**：根因是总时长把「最末片段终点」计入，拖长片段会反推总时长变大、整条轴随之重新缩放形成自反馈（音频时长未知时尤甚）。现拖动期间**冻结总时长**、并以轨道真实渲染宽度锁定「像素↔秒」换算，拖动与光标 1:1，比例全程稳定。
- 🩹 **ResizeObserver 自反馈**：放大视口测量与可缩放内容彻底解耦（固定宽度外层视口 + rAF 合并量取），杜绝缩放轨道把容器越撑越大。
- 🩹 **弹窗关闭后残留**：弹窗 `destroy-on-close` + 关闭即复位缩放，刻度数量封顶，避免异常状态泄漏到关闭后的视图。

### 说明
- 后端沿用既有片段契约（`start / end / model_id / model_ids`），时间轴的越界、空隙（间奏自动补原始近静音人声）、重叠（先到先得裁剪、不重复计算）均由现有管线统一归一化处理。
- 因片段与歌词解耦，**切换歌曲后建议重新获取 / 导入歌词以重建片段**。

### 安装
- 下载并运行 `XB-SVCB-Setup.exe` 一键安装（约 2 GB，含分离 / so-vits-svc / RVC / 模型站组件）。
- GPU 加速需 NVIDIA 显卡 + CUDA 12.1 运行库：https://developer.nvidia.com/cuda-12-1-0-download-archive
